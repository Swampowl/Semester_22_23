Zur Teilnahme an dieser Lehrveranstaltung benötigen Sie einen Computer und Sie müssen auf diesem Computer die nachfolgenden Softwarepakete installieren. Bitte installieren Sie diese Softwarepakete, wenn möglich, schon vor Vorlesungsbeginn! 


Hardware-Voraussetzungen

Sie benötigen einen handelsüblichen Computer mit Windows- oder MacOS-Betriebssystem.

Java 8, JDK (Programmiersprache)

Quelle: https://adoptopenjdk.net/releases.html?variant=openjdk8&jvmVariant=openj9
Für Windows: Download
Für MacOS: Download
Mit dem Download erhalten Sie einen Installer, mit dem die Software in einem Dialog eingerichtet wird. Verwenden Sie die Standardeinstellungen.


Eclipse IDE for Java Developers (Entwicklungsumgebung)

Quelle: https://www.eclipse.org/downloads/packages/installer
Für Windows: Download
Für MacOS: Download
Java 8 muss bereits installiert sein, bevor Sie die Eclipse IDE installieren. Mit dem Download erhalten Sie einen Installer. Nach dem Start des Installers werden Sie gefragt, welches Eclipse Sie installieren wollen. Wählen Sie: Eclipse IDE for Java Developers

 

Datenbanktreiber MariaDB/MySQL

https://mariadb.com/kb/en/installing-mariadb-connectorj/

 

VMWare-Player (Virtualisierungs-Software)

Diese Software benötigen wir in der zweiten Hälfte der Veranstaltung.
Quelle: https://www.vmware.com/de/products/workstation-player/workstation-player-evaluation.html
Für Windows: Download
Leider gibt es diese Software kostenlos nur für Windows. Für MacOS gibt es als Alternative lediglich die kostenpflichte Software "Fusion": https://www.vmware.com/de/products/personal-desktop-virtualization.html


Das VM-Image für Verteilte Anwendungen

Für diese Veranstaltung gibt es ein VM-Image, das Sie unter folgender Adresse finden.

https://bwsyncandshare.kit.edu/s/iKSfQmroZjASqqL

Beachten Sie bitte, dass diese Datei eine Größe von 4,3GB hat. Entpacken Sie diese Datei an einer beliegigen Stelle auf Ihrem Filesystem.

 

Alternative für MacOS-User: Virtual-Box

Freundlicherweise zur Verfügung gestellt von Mike Blank.

Für MacOS gibt es keinen kostenlose Version des VMWare-Players. Auf dem MacOS-Computer können Sie stattdessen die kostenlose Software Virtual-Box verwenden.

https://www.virtualbox.org/

Wenn Sie Virtual-Box verwenden, dann benötigen Sie das VM-Image in einem anderen Format. Verwenden Sie dafür folgendes VM-Image:

https://bwsyncandshare.kit.edu/s/26GNxFb7tKkQo4z
