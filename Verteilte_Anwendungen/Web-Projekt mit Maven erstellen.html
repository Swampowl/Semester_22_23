In Eclipse ein dynamisches Web-Projekt erzeugen

  • File - New - Dynamic Web Project - Project Name: DemoWebApp
  • Unter "Target Runtime" muss "Apache Tomcat v9.0" stehen. Wenn das nicht vorhanden ist, dann muss das mit "New Runtime" erst eingerichtet werden.
  • Dynamic web module version: 4.0
  • next
  • next
  • Finish

Den Deploymoment Descriptor erzeugen:

  • Projekt DemoWebApp auswählen - rechte Maustaste - Java EE Tools - Generate Deployment Descriptor

Das Projekt in ein Maven-Projekt umwandeln:

  • Projekt DemoWebApp auswählen - rechte Maustaste - Configure - Convert to Maven Project - Finish

Den Context Root festlegen:

  • Projekt DemoWebApp auswählen - rechte Maustaste - Properties - Web Project Settings - Context root folgendes eintragen: / - Apply and Close - OK

Die Datei web.xml (Deployment-Descriptor) finden Sie im Projekt unter Deployed Resources - webapp - WEB-INF. Öffnen Sie diese Datei. Fügen Sie nach dem welcome-file-list-Block den folgenden Text ein:

    <servlet>
        <servlet-name>javax.ws.rs.core.Application</servlet-name>
    </servlet>
    <servlet-mapping>
        <servlet-name>javax.ws.rs.core.Application</servlet-name>
        <url-pattern>/rest/*</url-pattern>
    </servlet-mapping>
    <session-config>
        <cookie-config>
            <http-only>true</http-only>
            <secure>true</secure>
        </cookie-config>
    </session-config>
